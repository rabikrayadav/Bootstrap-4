Grid System

The grid system is responsive, and the column will re-arrange automatically depending on the screen size.
Bootstrap allows up to 12 column acress the page.
If we do not want to use all 12 columns individually, we can group the columns together to create wider columns.

    Grid Classes

  • .col-
    • (extra small devices - screen width less than 576px)
  • .col-sm-
    • (small devices -screen width equal to or greater than 576px)
  • .col-md-
    • (medium devices - screen width equal to or greater than 768px)
  • .col-lg-
    • (large devices - screen width equal to or greater than 992px)
  • .col-xl
    • (Extra large devices -screen width equal to or greater than 1200px)

Responsive Image Classes

1) The .rounded class adds rounded corner to an image.
2) The .rounded-circle class shapes to the image to a circle
3) The .img-thumbnail class shapes the image to a thumbnail(bordered)
Float an image to the right with the .float-right class or to the left with .float-left
Create responsive images by adding
4) The .img-fluid class to the img tag. The image will then scale nicely to the parent element.
The .img-fluid class applies max-width:100% and height: auto; to the image.

Card Classes

1) .card class a basic card is created with this class
2) .card-body class add the content inside the card
3) .card-header class adds a heading to the card
4) .card-footer class adds a footer to the card
5) .card-title class to add card titles to any heading element
6) .card-text class to remove bottom margin
7) .card-link class to adds a blue color to any link and hover effect.
8) .card-img-top class to add image top of the card body.
9) .card-img-bottom class to add image bottom of the card body
10) .card-img-overlay class to add text on top of the image.
